# Context-Aware Temporal Knowledge Graph Embedding

## 背景   

   现有的方法将相关事实视为时不变的，忽略了相应的有效时段。实际上，许多关系是随着时间的推移而变化和涉及的，也就是说，它们只在一定的时间内有效。例如，关系实例  **配偶(“布拉德皮特”，“安吉丽娜朱莉”)**仅在时间间隔“[2014,2016]”内成立。
